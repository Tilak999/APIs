### Use telegra.ph as image hosting.
  http://telegra.ph/upload <br/>

  name = name of the file (anything)

  https://www.jsonstore.io/get-token <br/> 
  returns: TOKEN <br/>
  Example of end-point : https://www.jsonstore.io/(TOKEN)
